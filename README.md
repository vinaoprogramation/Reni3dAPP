# Reni3dAPP

Projeto criado como TCC. 
O objetivo pricipal é criar a versão Mobile de um sistema já funcional.
  O sistema prévio é um gerenciador de toda a extensão das impressões 3d no colégio Reni Correia Gamper, que manipula desde as máquinas até a aspecto financeiro. Para facilitar e agilizar sua utilização, o aplicativo foi criado com o objetivo de utilizar recursos nativos, como notificações, para otimizar sua experiência. Criado em React-Native com expo, o sistema se baseia no consumo de uma API do sistema original. 
  Tecnologias utilizadas: React-Native, Expo, Node e Javascript.
